Parqet Connect API

Download OpenAPI Document json

Download OpenAPI Document yaml

The *Parqet Connect API* empowers developers to seamlessly integrate
external financial data and tools with the Parqet portfolio tracking
ecosystem.


      Key Capabilities

  * *User Integration*: Retrieve connection status and permissions for
    the authenticated user.
  * *Portfolio Management*: Programmatically create new portfolios and
    retrieve metadata for existing ones.
  * *Activity Syncing*: Push and pull investment activities. The API
    supports a wide range of transaction types (Buy, Sell, Dividends,
    Fees, Transfers) and automatically handles:
      o *Securities*: Identified via ISIN (Stocks, ETFs).
      o *Cryptocurrencies*: Identified via Ticker Symbols.


      Authentication

This API uses OAuth2. Access is granularly controlled via scopes:

  * |portfolio:read|: Read access to portfolio data.
  * |portfolio:write|: Write access to portfolio data (creating
    portfolios and activities).

This endpoint is used to create activities in a specified portfolio.
Each transaction from a broker should be translated to one activity
(e.g. Buy, Sell, TransferIn, TransferOut).
